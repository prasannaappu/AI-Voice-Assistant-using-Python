
# AI Voice Assistant using Python

This is a Python-based desktop voice assistant that can perform tasks like:
- Playing YouTube music
- Sending WhatsApp messages
- Performing Google searches
- Taking screenshots

## Technologies Used
- Python
- Tkinter (GUI)
- pyttsx3 (Text-to-Speech)
- pywhatkit
